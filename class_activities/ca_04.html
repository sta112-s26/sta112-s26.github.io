Class activity: digging in to ggplot

In this activity, you will practice using ggplot to create and customize visualizations. You will use the penguins data from last week and todays class. Work in groups to answer the questions below. Solutions are provided for the first two questions, in case you get stuck.

Instructions:

  • Spend the remainder of the class period on this activity
  • You are welcome to work with a neighbor, but everyone must submit their own file to Canvas
  • When you are finished, render the file as an HTML and submit the HTML to Canvas (let me know if you encounter any problems)

Setup

Template file: ca_04_template.qmd

  • Rename the file to ca_04_your_name.qmd (so, for example, I would call it ca_04_ciaran_evans.qmd)
  • In the YAML, change author from Your Name to your actual name (keep the quotes)

The penguins data comes from the palmerpenguins package, and ggplot is included in the tidyverse package. You will need to load these packages (with library) to run the code today. If you use the Quarto template, those packages are included in the setup chunk, so you just need to run the setup chunk first.
